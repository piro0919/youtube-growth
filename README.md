# YouTube Growth

> AI-powered growth advisor for YouTube channels — data-driven strategies for audience and revenue.

[🔗 Live Site](https://youtube-growth.kkweb.io/)

## ✨ Features

- 🤖 AI-driven channel analysis
- 📊 Personalized growth recommendations
- 📈 Engagement and monetization insights
- 💾 PostgreSQL-backed data storage

## 🛠 Tech Stack

- Next.js + React + TypeScript
- Prisma (database ORM)
- Vercel Postgres
- Docker (local dev)

## 🚀 Development

```bash
# Local environment
docker-compose up
pnpm dev

# Database
pnpm migrate:create -- --name <name>
pnpm migrate:dev
pnpm studio:dev
```
